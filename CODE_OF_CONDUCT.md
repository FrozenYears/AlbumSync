# 行为准则 / Code of Conduct

本项目遵循 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 的简化版精神：

1. **就事论事**。围绕代码、文档、bug、需求展开讨论，不针对个人。
2. **尊重他人时间**。提 issue 前先搜索、自测；评审 PR 时给出具体可执行的反馈。
3. **拒绝骚扰**。任何形式的人身攻击、歧视性言论、骚扰行为都不可接受。

如遇违反，请通过 GitHub Issue 或站内私信联系维护者。维护者保留删除评论、关闭 PR、屏蔽账号的权利。

Full English text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
